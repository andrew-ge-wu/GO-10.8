The Greenfield Times example site

Greenfield Times is an example on how to build and structure a best practice web application project using Polopoly.

Maven is used to install and run the development environment. Run mvn -version to verify that Maven is installed on your system. Maven 3.0.3 or greater is required.

Classical Greenfield Times development using ant is also supported, and still required for production deployment.

Getting started and more information

See the Nitro Documentation available on http://support.polopoly.com/confluence/display/Polopoly1080/Nitro/ for information on how to get started.

For information on Polopoly specific Maven goals (running, deploying, etc), run the command 'mvn p:help' in the project root folder.

For in-depth information on how to install, configure and develop for Polopoly, see the developers guide (http://support.polopoly.com/doc/).
